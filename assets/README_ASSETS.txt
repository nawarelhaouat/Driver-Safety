Ce dossier contient les ressources multimédias du projet.

FICHIER AUDIO REQUIS : alarm.wav
---------------------------------
Ce fichier n'est pas inclus dans le dépôt.

Pour le générer automatiquement, exécutez depuis la racine du projet :
    python generate_alarm.py

Vous pouvez aussi placer ici votre propre fichier alarm.wav
(format recommandé : WAV mono ou stéréo, 44100 Hz, durée ~2s).

Sources gratuites :
- https://freesound.org  (rechercher "alarm beep" ou "car alarm")
- https://www.zapsplat.com
